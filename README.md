# 🤖 AI-Powered FAQ Chatbot

An intelligent FAQ Chatbot developed using Python, Natural Language Processing (NLP), TF-IDF Vectorization, Cosine Similarity, and Flask. The chatbot understands user queries, finds the most relevant FAQ, and provides accurate responses through a user-friendly web interface.

## 🚀 Features

* FAQ-based Question Answering System
* Natural Language Processing (NLP)
* Text Preprocessing with NLTK
* TF-IDF Vectorization
* Cosine Similarity Matching
* Flask Web Application
* Interactive Chat Interface
* JSON-based Knowledge Base
* Smart Response Matching
* User-Friendly Design

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/nishatschool55-boop/FAQ-Chatbot.git
cd FAQ-Chatbot
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
python app.py
```

Open in your browser:

```text
http://127.0.0.1:5000/
```

## 🎯 How It Works

1. User enters a question.
2. Text is preprocessed using NLP techniques.
3. TF-IDF converts text into numerical vectors.
4. Cosine Similarity compares the user's query with stored FAQs.
5. The most relevant answer is returned.
